SQL

I dette projekt har jeg skulle beskæftige mig med windows server, MS SQL database samt MS SQL server management. Ud over disse har jeg også skulle oprette diagrammer der beskriver databasen. Til selve serveren skulle jeg benytte hyper-v for at oprette en virtuel maskine hvorpå serveren lå. På serveren skulle jeg derefter oprette 3 brugere, installere SQL database og server management, hvortil jeg også skulle oprette 3 brugere. Til sql databasen oprettede jeg desuden et script som bestod af flere tabeller, der tilsammen skulle simulere en database over en skole. Herunder var elever, lærere, klasser og postnumre afbillede. Derudover skulle jeg importere et excel ark til en tabel, hvorfra jeg kunne finde postnumrenes tilsvarende bynavne. Desuden fik jeg også til opgave at mit script skulle kunne køre flere gange, uden at der opstår fejl, altså gøre mit script dynamisk.
